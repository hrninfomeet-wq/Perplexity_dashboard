## Quick Start

1. **Clone & Setup**
   ```bash
   git clone https://github.com/hrninfomeet-wq/Perplexity_dashboard.git
   cd Perplexity_dashboard
   ```

2. **One-Click Launch**
   ```bash
   .\start-project.bat
   ```

3. **Access Dashboard**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
